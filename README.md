# PathPilot Studio

A polished dark-mode student career planner that generates career roadmaps or suggests fitting paths based on current skills and available time.

## Generated with GenMB

This project was generated using [GenMB](https://genmb.com) - AI-powered application builder.

### Original Prompt

> Build a modern web app called "pathpilot" for students.

The app should help users choose a career path or generate a learning roadmap based on their current skills and available time.

The app must have two modes:

1. "I Know My Goal"
- Input fields:
  - Goal (dropdown: Web Development, Cybersecurity, Data Science)
  - Current skills (text input)
  - Time available per day (dropdown)
- Output:
  - Step-by-step roadmap
  - Recommended websites/resources
  - Estimated time to complete
  - Tips and what to avoid

2. "I Don't Know My Goal"
- Input fields:
  - Current skills
  - Time available
- Output:
  - 2–3 suggested career paths
  - Explanation why each path suits the user
  - Comparison between options

Design:
- Dark theme with purple accents
- Clean and modern UI
- Card-based layout
- Smooth animations
- Mobile responsive

Behavior:
- Show results on the same page (no page reload)
- Use simple logic (no backend required)
- Make it interactive using JavaScript

Make the UI professional and visually appealing.
